# Task Manager & Scheduler

A command-line task management system built in C++ to organize daily tasks with priorities, deadlines, and persistent file storage.

---

## Features

* Add, edit, and delete tasks
* Mark tasks as completed
* Search tasks by keyword
* Filter tasks by priority
* Sort tasks by deadline
* Due date alerts
* Statistics dashboard
* Automatic file save/load
* Color-coded console interface
* Input validation for priority and deadline

---

## Concepts Used

* Object-Oriented Programming
* STL Vector
* File Handling (`fstream`)
* Lambda Functions
* Sorting with `std::sort`
* String Processing
* Input Validation
* Date Parsing
* ANSI Escape Codes

---
