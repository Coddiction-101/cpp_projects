## Overview
A tool that scans a target folder and automatically sorts files into categorized subfolders based on their type. It detects duplicates, generates an organization report, and supports undoing the last operation.

## Features
- Auto-sorts files into categories: Images, Videos, Documents, Code, Audio, Archives, Others
- Detects and reports duplicate files
- Generates an organization summary report
- Undo last organization action
- Cleans file names by removing spaces and special characters
