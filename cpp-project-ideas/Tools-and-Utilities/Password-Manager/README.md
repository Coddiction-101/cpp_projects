## Overview
A secure, console-based vault to store, encrypt, and retrieve passwords associated with websites or apps. Features include strong password generation and master password protection.

## Features
- Store passwords linked to website/app names
- Encrypt stored passwords (XOR/Caesar cipher)
- Protect access with a master password
- Generate strong, customizable passwords
- Search for passwords by name
- Copy passwords to clipboard
- Persist encrypted data in files
