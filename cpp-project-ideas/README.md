## 🏆 Completed Projects

### ✅ Task Manager & Scheduler
> A console-based task management system with color-coded priorities, deadline alerts, search, sort, and persistent storage.
- **Concepts:** OOP, Vectors, File I/O, ANSI colors, Lambda sort, Date parsing
- **Version:** v2.0
- **Repo:** [View Code](https://github.com/Coddiction-101/cpp_projects/tree/main/TaskManager%26Schedular)

### ✅ Banking System - ATM Simulator
> A full ATM simulator with account management, PIN authentication, transfers, and transaction history.
- **Concepts:** OOP, Map, Pointers, File I/O, Authentication
- **Version:** v1.0
- **Repo:** [View Code](https://github.com/Coddiction-101/cpp_projects/tree/main/BankingSimulation)

### ✅ Terminal Text Editor
> A fully functional terminal text editor with search, undo/redo, scrollbar, auto-indent, copy/paste, and file open/save.
- **Concepts:** Raw input, Win32 API, ANSI codes, Snapshot-based undo/redo, Viewport scrolling
- **Version:** v2.1
- **Repo:** [View Code](https://github.com/Coddiction-101/cpp_projects/tree/main/TerminalTextEditor)
