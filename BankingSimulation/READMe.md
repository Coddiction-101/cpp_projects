# Banking System (ATM Simulator)

A console-based banking application built in C++ that simulates basic ATM operations, including account management, secure transactions, and persistent data storage.

**Category:** Tools & Utilities
**Status:** Complete

---

## Features

* Create Savings and Current accounts
* PIN-based authentication
* Deposit and withdraw money
* Transfer funds between accounts
* View account details
* Transaction history
* Delete accounts with confirmation
* Automatic file save/load
* Input validation and error handling

---

## Concepts Used

* Object-Oriented Programming
* STL Map
* STL Vector
* File Handling (`fstream`)
* Pointers & References
* Authentication Logic
* Menu-Driven Application Design

---

## Project Structure
